# CombatMarket

CombatMarket is a platform connecting combat sports fighters with fans through curated product recommendations.

## Technology Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

To run locally:

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment

Deploy via the project dashboard or connect a custom domain through Settings > Domains.
